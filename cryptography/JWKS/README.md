# JWK/JWE Toy

Playing with node-jose for JWKs and JWE.

## Usage
```bash
npm init
npm install node-jose

node jwks.js help
node jwks.js <command>
```

## Notes

- The help command outputs the available commands
- Key stored in keystore.json (ignored)
- Useful RFCs to read are https://www.rfc-editor.org/rfc/rfc7518 and https://www.rfc-editor.org/rfc/rfc7517
- node-jose: https://github.com/cisco/node-jose
